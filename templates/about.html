About

What is this site?

This is an AI-powered book recommender, developed for the BBCS2023 Advanced Course Hackathon.
Our Team: Li Run, Rongyi, Sai


User Guide:

Select any of the books stored in our database via the input box provided.
The AI will then recommend a list of books that are similar in content to the input book.


W.I.P/Potential Improvements:

  • More parameters to be considered in recommending books
  • Adjustable number of books in output
  • Adjusting how similar the user wants the recommended books to be
